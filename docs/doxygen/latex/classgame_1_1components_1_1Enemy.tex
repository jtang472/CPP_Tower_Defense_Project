\doxysubsection{Detailed Description}
base class for all types of enemies in the game 

in the current design, we do not necessarily need any derived class for enemies. Instead, all enemy behavior should be defined in the Enemy\+Define json file. So this class should be purely data driven and should be capable of handling any enemy behavior defined in the json data file
