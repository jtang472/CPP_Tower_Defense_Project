\doxysubsection{Detailed Description}
base class for all types of towers in the game 

in the current design, we do not necessarily need any derived class for towers. Instead, all tower behavior should be defined in the Tower\+Define json file. So this class should be purely data driven and should be capable of handling any tower behavior defined in the json data file
