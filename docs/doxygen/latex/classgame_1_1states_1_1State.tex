\doxysubsection{Detailed Description}
base class of all game states 

this is the base class of all game states. A game state controls the game flow, and it allows a state to be easily reused and stack on each other to create multiple states for example, a game pause state can be stack on the game state so the game state can be resumed once the game pause state gets poped
