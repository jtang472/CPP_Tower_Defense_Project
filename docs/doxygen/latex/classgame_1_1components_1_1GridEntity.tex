\doxysubsection{Detailed Description}
base class for all the drawables in the game 

this is the base class for every game entity that can be put into the grid system (towers, enemies, projections...) we ensure a one-\/way dependency chain of grid system -\/$>$ grid layer -\/$>$ grid -\/$>$ grid entity
