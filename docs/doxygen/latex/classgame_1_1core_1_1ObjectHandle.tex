\doxysubsection{Detailed Description}
\subsubsection*{template$<$class TObject$>$\newline
class game\+::core\+::\+Object\+Handle$<$ TObject $>$}

object reference handle 

this class is an alternative / more efficient solution compared to std\+::shared\+\_\+ptr \& std\+::weak\+\_\+ptr, instead of storing the actual object reference, this class caches the object guid and the index of the object in a global lookup table. To get the actual object reference from the object handle, it query the global look up table and compare the two guids. If guids are equal, then the object handle is still valid and we return object reference to caller, otherwise the object is invalid and we return nullptr
