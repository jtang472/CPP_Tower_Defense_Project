\doxysubsection{Detailed Description}
base class for all components 

this is the base class for all derived components. It defines a series of life cycle functions that gets called at a certain point of a component\textquotesingle{}s lifetime. All components are managed by gameobjects. lifetime design\+: construct -\/$>$ onenable -\/$>$ tick$^\wedge$$\ast$ -\/$>$ ondisable -\/$>$ destruct component can be enabled and disabled before destruct to restart its lifetime
