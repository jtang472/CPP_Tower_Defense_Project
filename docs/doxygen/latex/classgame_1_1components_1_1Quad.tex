\doxysubsection{Detailed Description}
drawables to make use of the vertex array provided by SFML 

currently, this class is only used with tiles in the grid system to eliminate black artifacts at the edge of the tile, if you want to draw normal GUI elements, use \mbox{\hyperlink{classgame_1_1components_1_1Sprite}{Sprite}} class instead
