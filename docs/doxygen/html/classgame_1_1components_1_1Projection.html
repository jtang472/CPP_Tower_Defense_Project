Detailed Description

base class for all types of projections in the game

in the current design, we do not necessarily need any derived class for projections. Instead, all projections behavior should be defined in the ProjectionDefine json file. So this class should be purely data driven and should be capable of handling any projection behavior defined in the json data file

Zheng Yang
