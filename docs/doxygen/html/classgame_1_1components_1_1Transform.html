Detailed Description

transform component of a gameobject

this class defines the size, position, rotation and scale of a gameobject in the game world it also defines a hierarchy system of gameobjects - a transform can be set to be the parent of other transforms, this set the origin of the local positions of its child to be the position of the current transform
